# VUTTR

This is a simple api that allows to manage tools with its name, url, description and tags.

## Getting started

https://vuttr4.docs.apiary.io/

## How to use

To initialize the api

```
yarn init

```

To start the api in dev

```
yarn dev
```

To start the api in production

```
yarn start
```

To test the api

```
yarn cypress open
```
